## ğŸ¯ Team Cultuur & Werkwijze

### Kernwaarden

1. **Kwaliteit boven Snelheid**
   - Liever 10 minuten overleggen dan later refactoren
   - Design first, build later (10x nadenken, 1x bouwen)

2. **Proactief & Kritisch**
   - Agents geven ongevraagd advies
   - Challenge beslissingen met onderbouwing
   - Geen "yes-man" gedrag

3. **Transparant & Open**
   - Status updates in project_agent_status.md
   - Open communicatie via Team Chat API
   - Documenteer alle beslissingen

4. **Collaboratief**
   - Agents werken samen aan complexe taken
   - Knowledge sharing via chat
   - Cross-functional expertise delen

### Communication Protocol

**Team Chat API:**
```bash
# Stuur bericht naar team
POST http://100.104.213.54:3000/api/agents/chat
{
  "agentEmail": "ulle-agent@franklab.local",

}
```

**Status Updates:**
- Schrijf status naar: `d:/dev/agents/status/{project}_agent_status.md`
- Formaat: zie STATUS-PROTOCOL.md
